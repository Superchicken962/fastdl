Now that you have sent an email to add your files, you should now set up your server so the files actually get downloaded.

We are going to start, we assume you have set up your server and have it ready to go.

Steps
1. Go to "\tf\cfg" in your server directory
2. Open up server.cfg in your favourite text editor (or any that work)
3. Add these 3 lines to your server.cfg:

sv_downloadurl "https://freefastdl.cf/fastdl/tf/"
sv_allowdownload 1
sv_allowupload 1

4. Congrats! You have set up your server to have our fastdl server!
Pretty easy hey?
5. If you need to upload more files just go back to the other page and keep sending emails! Try to send files in bulk, not all in different emails.
